# Basic CSS Syntax

## Structure:

```css
selector {
  property: value;
  property: value;
}
```

- **Selector**: Targets the HTML element.
- **Property**: What you want to change.
- **Value**: The setting you apply to the property.

Example:

```css
p {
  color: blue;
  font-size: 16px;
}
```
